The project is about a bank registration model which has different models ranging from registration to displaying all the data.

          Agile Methodology:
Agile Methodology has some basic terminologies:
These include:
1)Tasks
2)User Stories
3)Epics
4)Sprints
5)Product backlog

 Tasks,
These are the basic functionalities needed in a model which the developers have to design in order to support user requirements like Name,Id or anything else.
These are the smallest units in the web working models.

Example of banking model,
REGISTRATION FORM
registration id,Password,Account type,Account Number form some of the various tasks in the agile methodology.These are defined on the basis of user stories.
Specific styles,styles are added to give a good look to the website.

User Stories,
These are the various specifications that the user wants.These are taken into consideration while creating the model.These form a vital part in the development process.
The tasks are defined on the basis of various user stories.
